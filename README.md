# Guess The Number Game

A simple Python game where the computer generates a random number between 1 and 100, and the user tries to guess it.

## Features

- Random number generation
- Unlimited guessing attempts
- Hints ("Too high" or "Too low")
- Displays the total number of attempts

## Technologies Used

- Python 3
- random module

## How to Run

1. Install Python.
2. Download or clone this repository.
3. Run the following command:

```bash
python guess_the_number.py
```

## Example Output

```

Enter your guess (1-100): 50
Too low! Try again.

Enter your guess (1-100): 75
Too high! Try again.

Enter your guess (1-100): 68
Congratulations! You guessed the number in 3 attempts.
```
